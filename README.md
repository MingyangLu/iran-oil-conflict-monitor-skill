# Iran Oil Conflict Monitor

`iran-oil-conflict-monitor` 是一个面向原油市场场景的情报分析 skill，用来追踪美伊冲突、霍尔木兹海峡通行、特朗普和美国官方表态、伊朗高层表态，以及这些事件对 `Brent`、`WTI` 等原油价格的影响。

它的目标不是做泛泛的新闻摘要，而是把高频地缘政治信息整理成更接近投研晨报或交易台 briefing 的输出，强调事实核验、时间戳、来源质量和油价传导链条。

## 适用场景

- 查询过去 24 至 72 小时美伊冲突的最新进展
- 跟踪霍尔木兹海峡油轮、LPG 船、航运延误和通行恢复情况
- 检索特朗普关于伊朗、中东、油价的最新发帖或公开表态
- 核验伊朗最高领袖、总统、外交部、议长等高层口径
- 评估事件对 `Brent`、`WTI`、航运风险溢价的影响
- 输出固定格式的投研晨报，并附带利多/利空量化评分

## 核心能力

- 优先使用官方来源和主流媒体进行双重核验
- 将特朗普社交平台内容纳入检索链路，优先看 `Truth Social`
- 区分“真实供给冲击”和“仅风险溢价交易”
- 输出固定晨报模板，减少风格漂移
- 提供 `-5` 到 `+5` 的油价影响评分，并映射为“强利多 / 偏利多 / 中性 / 偏利空 / 强利空”

## 信息源优先级

1. 美国和伊朗官方页面
2. 特朗普原始发帖平台和可核验转载
3. 航运与海峡通行相关数据或权威媒体报道
4. Reuters、AP、Bloomberg、WSJ、FT 等市场相关确认来源
5. Brent / WTI 最新价格表现

## 输出内容

默认输出为固定格式的中文晨报，通常包含：

- 一句话结论
- 今晨核心变化
- 关键事实核验
- 利多/利空量化评分
- 评分拆解
- 市场影响分析
- 三种情景
- 风险提示
- 信息来源

## 评分逻辑

初始分数为 `0`，再根据以下维度加减分：

- 实际供给风险
- 霍尔木兹运输风险
- 美方政策与军事动作
- 伊方表态与行动
- 市场是否已经提前计价

最终分数限制在 `-5` 到 `+5` 之间，并映射为：

- `+4` 到 `+5`：强利多
- `+2` 到 `+3`：偏利多
- `-1` 到 `+1`：中性
- `-2` 到 `-3`：偏利空
- `-4` 到 `-5`：强利空

## 示例用法

```text
用 $iran-oil-conflict-monitor 查询最近48小时美伊冲突动态，重点看霍尔木兹油轮通过量、白宫表态和伊朗最高领袖讲话，并分析对布油和WTI的影响。
```

```text
用 $iran-oil-conflict-monitor 做一份今天的投研晨报，加入利多/利空量化评分，并说明特朗普最近的 Truth Social 发帖是否影响油价。
```

```text
用 $iran-oil-conflict-monitor 查过去24小时特朗普关于伊朗的最新发帖，优先找 Truth Social 原帖，再判断是否同步到了 X，并分析对油价的影响。
```

## 设计原则

- 先核验，再判断
- 先找原始表态，再看媒体解读
- 先分清运输中断还是情绪交易，再谈油价方向
- 输出要接近投研或交易台能直接使用的格式
